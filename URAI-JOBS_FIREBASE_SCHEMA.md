# URAI-JOBS FIREBASE SCHEMA

This document outlines the Firestore schema for the URAI-JOBS project.

## Collections

*   **users:** Stores user account information.
*   **candidateProfiles:** Stores candidate profile information.
*   **employerProfiles:** Stores employer profile information.
*   **jobs:** Stores job listings.
*   **applications:** Stores job applications.
*   **messages:** Stores messages between users.
*   **notifications:** Stores user notifications.
