# URAI-JOBS

This is a Firebase-backed job/orchestration service for URAI Labs.

## Local Development

1.  **Install Dependencies:**
    ```bash
    pnpm install
    ```

2.  **Run Emulators & Web App:**
    ```bash
    pnpm run dev
    ```

3.  **Seed the Database (in a separate terminal):**
    ```bash
    pnpm run seed
    ```

## Deployment

```bash
firebase deploy
```
