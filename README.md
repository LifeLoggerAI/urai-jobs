# URAI-Jobs

This project is a hiring + applicant tracking + waitlist + referrals system for URAI Labs / URAI.

## Local Setup

1. Install Firebase CLI: `npm install -g firebase-tools`
2. Install project dependencies: `pnpm install`

## Running the Emulators

To run the Firebase emulators, use the following command:

```bash
firebase emulators:start
```

## Seeding the Database

To seed the database with test data, run the following command:

```bash
npm run seed
```

## Running the Web App

To run the web app, use the following command:

```bash
cd web && npm run dev
```

## Deployment

To deploy the project, use the following command:

```bash
firebase deploy
```
