# URAI-JOBS Deployment

This document describes how to deploy the URAI-JOBS project.

## Prerequisites

- Firebase CLI
- Node.js
- pnpm

## Deployment Steps

1. **Install dependencies:**

   ```bash
   pnpm install
   ```

2. **Build the project:**

   ```bash
   pnpm build
   ```

3. **Deploy to Firebase:**

   ```bash
   firebase deploy
   ```
